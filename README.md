# Voice Agents Documentation

Official documentation for Voice Agents - AI-powered conversational systems that automate voice interactions at scale.

## About Voice Agents

Voice Agents combine advanced natural language processing, speech recognition, and intelligent call flow management to handle customer communications without human intervention. Our platform enables businesses to:

- Automate customer support and sales calls
- Integrate with existing systems (Shopify, WhatsApp, Klaviyo, etc.)
- Scale to millions of concurrent calls
- Pay only for productive conversation time

## Documentation Structure

- **Getting Started** - Overview, campaign management, and FAQ
- **Guides** - Detailed tutorials for specific use cases
- **Integrations** - Setup instructions for connecting your tools
- **Use Cases** - Industry-specific applications
- **API Reference** - Developer documentation and endpoints

## Local Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) to preview documentation changes locally:

```bash
npm i -g mint
```

Run the following command at the root of your documentation:

```bash
mint dev
```

View your local preview at `http://localhost:3000`.
